# Amazon Data Analysis

## 📌 Project Overview
This project analyzes Amazon product data using Python. It includes data cleaning, exploratory data analysis (EDA), and data visualization to discover insights about product prices, discounts, ratings, and customer reviews.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset
- Amazon Product Dataset (`amazon.csv`)

## 📊 Analysis Performed
- Data Cleaning
- Missing Value Handling
- Duplicate Value Check
- Data Type Conversion
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Heatmap

## 📈 Visualizations
- Rating Distribution
- Top 10 Product Categories
- Top 10 Most Reviewed Products
- Average Discount by Category
- Price vs Rating Scatter Plot
- Correlation Heatmap

## 🔍 Key Insights
- Products with higher discounts do not always receive higher ratings.
- Discounted price and actual price are strongly correlated.
- USB cables and electronic accessories are among the most common product categories.
- Ratings have only a weak relationship with product prices.
